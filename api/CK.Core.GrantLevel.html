    Enum GrantLevel

    Standard grant level: this is a simple (but often enough) way to secure a resource. Among the different levels, depending on the actual resource, some (or most) of them are useless and can be ignored. But for some kind of resources all of them make sense: a "service object" (a kind of executable process) for instance can benefits of all these levels.
