    CovariantMatch(Type, Type)

    Checks whether a type is "above" (or "more general than") another one. This check uses System.Type.IsAssignableFrom(System.Type) when the main type is not generic. For generics (be it an interface or a class), inheritance relations are used recursively on generic parameters: the "is assignable from" semantics is lost in favor of a more relaxed relation, for instance IEnumerable[ICollection[ValueType]] "Covariant matches" with IEnumerable[IList[int]].
