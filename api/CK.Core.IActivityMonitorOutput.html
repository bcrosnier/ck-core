    RegisterUniqueClient<T>(Func<T, Boolean>, Func<T>)

    Registers a IActivityMonitorClient that must be unique in a sense.

    Remarks

    The factory function MUST return a client that satisfies the tester function otherwise a System.InvalidOperationException is thrown. The factory is called only when the no client satisfies the tester function: this makes the 'added' out parameter useless.
