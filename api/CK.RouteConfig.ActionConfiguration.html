    Name

    Gets or sets the name of this configuration. Can not be null.

    Remarks

    Whether this name is a valid one or not is not checked by the configuration itself but by the MetaConfiguration that "covers" it: different rules may be implemented for the name of a configuration depending of the way it is used, this is the role of the meta configuration.
