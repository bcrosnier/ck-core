    MinimalFilter

    Gets or sets the minimal filter for this handler. Defaults to Undefined: unless specified, the handler will not participate to ActualFilter resolution of monitors that eventually are handled by this handler.See remarks.

    Remarks

    This is NOT a filter: this is the minimal filter that guaranties that, at least, the specified levels will reach this handler.

    A concrete handler can, if needed, define a true filter: it is its business to retain or forget what it wants.
