    FormatWhenUniquifier

    @"{0:yyyy-MM-dd HH\hmm.ss.fffffff}({1})" is the format that will be used to format log time when the Uniquifier is not zero. It is based on FileNameUniqueTimeUtcFormat (that is used as-is when the Uniquifier is zero) for the date time format.
