    CK.Core

    The CK.Core assembly (NuGet package: CK.Core) contains core utility classes and interfaces, both static and non-static, used by other components.

    CK-Core (with a dash) is the name of the complete set of components containing CK.Core, CK.ActivityMonitor, CK.Monitoring, etc.
    CK.Core (with a dot) is the name of this particular assembly and package.

    Installing CK.Core

    To install CK.Core, install the NuGet package CK.Core in your .NET library or application.

    Compatibility

    CK.Core is compatible with projects targeting the .NET framework 4.5.1 or above.

    Using CK.Core

    This package contains a number of various utilities used in other components and libraries, grouped in coincidental cohesion, and does not come with one primary usage or purpose.

    See below for the detailed contents of the package.

    Contents

    Configuration and application settings

    AppSettings

    The AppSettings class is a simple front to access AppSettings through DI-like calls, with eg. an AppSettings.Default singleton and methods like T GetRequired<T>(string key).

    Binary reading/writing

    CKBinaryReader

    The CKBinaryReader class specializes System.IO.BinaryReader with helper methods supporting nullable strings and compressed integers.

    CKBinaryWriter

    The CKBinaryWriter class specializes System.IO.BinaryWriter with helper methods supporting nullable strings and compressed integers.

    Exception handling and formatting

    CKException

    The CKException class specializes System.Exception with serialization and helpers for text formatting and display.

    CKExceptionData

    The CKExceptionData class is an immutable, serializable representation of another System.Exception. Used by CKException.

    CriticalErrorCollector

    The CriticalErrorCollector class is a thread-safe collector of System.Exception instances, raising events when errors are added.

    Collections

    CK.Core defines the following collection-related interfaces:

    • ICKReadOnlyCollection<T> - Contravariant interface inheriting IReadOnlyCollection<T>
    • ICKReadOnlyList<T> - Covariant interface inheriting IReadOnlyList<T>
    • ICKReadOnlyUniqueKeyedCollection<T, TKey> - Read-only, keyed collection of covariant items with a contravariant key
    • ICKReadOnlyMultiKeyedCollection<T, TKey> - Read-only, keyed collection of covariant items with a contravariant key, supporting multiple items having the same key
    • ICKWritableCollection<T> - Contravariant interface of a writable collection of items (with Add(), Remove(), Clear(), Count)
    • ICKWritableCollector<T> - Contravariant interface of an item collector (with Add(), Count)

    CKEnumeratorMono<T>

    The CKEnumeratorMono<T> class is an implementation of IEnumerator<T> containing and optimized for a single element.

    CKReadOnlyCollectionOnICollection<T>

    The CKReadOnlyCollectionOnICollection<T> class is an adapter class exposing an IReadOnlyCollection<T> from an ICollection<T>.

    CKReadOnlyListOnIList<T>

    The CKReadOnlyListOnIList<T> class is an adapter class exposing an IReadOnlyList<T> from an IList<T>.

    CKSortedArrayList<T>

    The CKSortedArrayList<T> class is a collection class keeping the order given by an IComparer<T>.

    CKSortedArrayKeyList<T, TKey>

    The CKSortedArrayKeyList<T, TKey> class is a collection class keeping the order given by an IComparer<T>, where the order is not given from the item itself.

    FIFOBuffer<T>

    The FIFOBuffer<T> class is a queue with a fixed maximum size. Its oldest items are dropped when its maximum size is reached.

    Extensions

    CK.Core defines the following collection-related extension classes in namespace CK.Core:

    • CKReadOnlyExtension
    • CollectionExtension
    • DictionaryExtension
    • EnumerableExtension

    Logging-related utilities (CKTrait, DateTimeStamp)

    CKTrait

    The CKTrait class is an immutable object associated to a string, which can be atomic (eg. "Alt"), or combined (eg. "Alt|Ctrl|Shift"). It is unique within a CKTraitContext.

    Used as log entry tags in logging utilities, or when describing key combinations.

    CKTraitContext

    The CKTraitContext class is a context containing CKTrait instances, used to register and make use of CKTrait instances.

    SetOperation

    The SetOperation enum describes the various operations possible on a set of items (Union, Intersect, etc.). Can be used on eg. CKTrait instances to perform combination operations.

    DateTimeStamp

    The DateTimeStamp class encapsulates a DateTime value with another value, unique per DateTime value, called a Uniquifier.

    In logging, this safeguards the logging order for multiple log entries with the same DateTime, giving the entry a unique key and making it potentially addressable.

    CK.Core defines a DateTimeStamp-related extension class in namespace CK.Core: DateTimeStampExtension

    Files and input/output

    FileUtil

    The FileUtil static utility class contains file-related static properties and methods, such as:

    • CompressFileToGzipFile(...) and CompressFileToGzipFileAsync(...)
    • CheckForWriteAccess(...)
    • CopyDirectory(...), supporting file filters
    • NormalizePathSeparator(...)
    • Utilities to create and make use of uniquely-timed files, when creating timestamped files (eg. logs), ensuring they can be created.

    TemporaryFile

    The TemporaryFile class is a disposable class, which creates a temporary file when instanciated, and deletes it when disposed.

    Access control

    GrantLevel

    The GrantLevel enum describes generic access levels of any actor on any resource. Intended for use in access control systems.

    Reflection and debugging

    SimpleTypeFinder

    The SimpleTypeFinder class is a static utility class to resolve Types from their namespace and name, but without the complete AssemblyQualifiedName of their assembly.

    Dependency injection

    ISimpleServiceContainer

    The ISimpleServiceContainer interface describes a dependency injection container with the ability to add and remove services.

    CK.Core defines an ISimpleServiceContainer-related extension class in namespace CK.Core: ServiceContainerExtension.

    SimpleServiceContainer

    The SimpleServiceContainer class is a disposable implementation of ISimpleServiceContainer.

    XML

    CK.Core defines an XML-related extension class in namespace CK.Core: XmlExtension.

    Miscellaneous interfaces and static utilities

    IMergeable

    The IMergeable interface describes an object supporting merging with another object.

    IUniqueId

    The IUniqueId interface describes an object having a unique Guid.

    Util

    The Util static utility class contains miscellaneous static methods and properties:

    • Empty singletons for Enumerable, Array (pre-.NET 4.6), IDisposable, Action
    • Binary search (Wikipedia: Binary search algorithm) implementation methods
    • Thread-safe (Interlocked) utility methods to perform Add, Remove, RemoveAll and Set operations on an Array
    • Thread-safe (Interlocked) utility method to set a reference type to a variable

    Util.Hash

    The Util.Hash contains static utility methods to combine hash values.
